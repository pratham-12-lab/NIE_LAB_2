Q1. String rotation based on squares

You are given a list of strings each having 2 components to it - word and integer. These 2 components are separted by :

  • If the square of digits in the number component is even, then you must rotate the word to the right by 1.
  • If the square of digits in the number component is odd, then you must rotate the word to the left by 2.

Finally, print the rotated word.

Example Input:
rhdt:246, gftd:1246
Output:
trhd
ftdgh
Explanation: 
2*2+4*4+6*6 = 56 which is even so rotate rhdt -> trhd

1*1+2*2+4*4+6*6 = 57 which is odd so rotate ftdgh -> ghftd

Q8. Canvas Frames

Given n sticks whose lengths equals a1, a2, …. an.

And you want to make a h * w-sized frame, to make a frame you need two sticks whose lengths equal w. Specially, to make a square frame (when h = w),you needs four sticks of the same length.

You have to calculate, what is the maximum number of frames that can be made using the given sticks. The area of each frame doesn’t matter. We want the most number of frames possible.

Note: It is not necessary to use all the sticks you have.

Example Input:
[2,2,3,2,3,2]
Output:
1
Explaination:
Either you can make a frame of size 2*3 or 3*2 or 2*2 but you can make only one frame because after making a frame of any of these size,
there are only 2 sticks left.

Q11. Distinct Numbers in Windows

You are given an array of N integers, A1, A2, …., AN and an integer K. Return the count of distinct numbers in all windows of size K.

Formally, return an array of size N-k+1 where i’th element in this array contains number of distinct elements in sequence Ai, Ai+1, ….. , Ai+K-1.

Note: If K>N, return empty array.

Example Input:
A = [1,2,1,3,4,3]
K = 3
Output:
[2,3,3,2]
Explaination
All windows of size K are
[1,2,1]
[2,1,3]
[1,3,4]
[3,4,3]
