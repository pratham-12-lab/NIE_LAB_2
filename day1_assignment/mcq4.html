Q1: Unique substring of given string

Given a string, we have to find the longest substring which is unique (that has no repetition) and whose size is at least 3.

If more than one substring is found with max length then we have to print which appeared first in the string.

If no substring is present that matches the condition then we have to print -1.

Example Input:
A@bcd1abx
Output:
A@bcd1a

Q2: Largest palindromic substring

Take as input a string. Find the longest possible substring from this string which is also a palindrome.

Finally, print the rotated word.

Example Input:
dcodocir
Output:
codoc
ftdgh

Q3: String rotation based on squares

You are given a list of strings each having 2 components to it - word and integer. These 2 components are separted by :.

  • If the square of digits in the number component is even, then you must rotate the word to the right by 1.
  • If the square of digits in the number component is odd, then you must rotate the word to the left by 2.

Finally, print the rotated word.

Example Input:
rhdt:246,gftd:1246
Output:
trhd
ftdgh
Explanation: 
2*2+4*4+6*6 = 56 which is even so rotate rhdt -> trhd

1*1+2*2+4*4+6*6 = 57 which is odd so rotate ftdgh -> ghftd

Q4: Consecutive numbers in matrix

You are given an m*n matrix such that n=m+1. In the given matrix, find if any number is consecutive for 3 times either in row, column, diagonals and print the num.

If there are multiple such numbers then print minimum of those numbers. Input format

  • First line contains m, the number of rows
  • Following m lines contain n numbers

Example Input:
6 
2 3 4 5 6 2 4
2 3 4 7 6 7 6
2 3 5 5 5 5 2
2 3 1 1 2 1 3
1 1 1 1 9 0 3
2 3 1 1 5 1 2
Output:
1
